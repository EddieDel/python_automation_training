# Excel Report Generator

A Python automation script that generates formatted Excel reports from employee data.

## Features
- Automatically creates Excel files from a list of employees
- Bold and color-highlighted headers
- Auto-sized column widths
- Total salary row at the bottom
- Reusable for any dataset with any filename

## Requirements
- Python 3.x
- openpyxl library

## How to Use

Install the required library:
```
pip install openpyxl
```

Run the script:
```
python excel_generator.py
```

## Example Output
Generates a formatted .xlsx file with Name, Department, and Salary columns, bold yellow headers, and a total salary row at the bottom.
