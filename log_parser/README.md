# Verification log file analyzer

A Python automation script that generates formatted Excel reports from synopsys vcs log file.

## Features
- Automatically creates Excel files from a log file.
- Bold and color-highlighted headers
- Auto-sized column widths
- Totals of each category
- Reusable for any log with any filename

## Requirements
- Python 3.x
- openpyxl library

## How to Use

Install the required library:
```
pip install openpyxl
```

Run the script:
```
python log_parser.py
```
