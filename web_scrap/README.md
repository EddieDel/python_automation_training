# Excel Webpage Scraper 

A Python automation script, that scrapes webpages and generates formatted Excel reports for books.

## Features
- Automatically creates Excel files from a list of books
- Bold and color-highlighted headers
- Auto-sized column widths
- Total cost of number of books at the bottom


## Requirements
- Python 3.x
- openpyxl library

## How to Use

Install the required library:
```
pip install openpyxl
pip install requests beautifulsoup4
```

Run the script:
```
python scraper.py
```

## Example Output
Generates a formatted .xlsx file with Title, Price and a total cost at the bottom.
